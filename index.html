Here is the copy/paste for Wilton: Hey Wilton! Here is how to push your portfolio to GitHub through the terminal every time you make changes. First time setup only. Do these once: Open VS Code, open your portfolio folder, click Terminal at the top then New Terminal. Run these one at a time: git init git add . git commit -m "first commit" git remote add origin https://github.com/YOURUSERNAME/YOURUSERNAME.github.io.git git branch -M main git push -u origin main When it asks for your username type your GitHub username. When it asks for your password do NOT type your actual password. Instead go to github.com, click your profile picture, click Settings, scroll all the way down and click Developer Settings, click Personal Access Tokens, click Tokens Classic, click Generate New Token, check the repo checkbox, click Generate Token, copy the long code it gives you and paste that in as your password. Save that token somewhere safe. You will need it every time. Every time you make changes after that just run these three: git add . git commit -m "describe what you changed" git push Wait 30 seconds then refresh your live site and your changes will be there. That is it!
